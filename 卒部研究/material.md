剛体振り子の運動であれば高校物理の範疇だが、紐（連続体）の運動となると様相は全く異なる。要求通り、この系を「$N$個の質点と微小な長さを持つ剛体棒からなる$N$重振り子」として定式化し、$N \to \infty$ の連続極限をとることで、初期状態における紐の特異な挙動を解析する。

---

### **1. N重振り子の定式化**

紐の全長を$l$、全質量を$M$とする。これを$N$等分し、長さ$a = l/N$、質量$m = M/N$の剛体棒が連なった$N$重振り子を考える。
固定端を原点とし、鉛直下向きに$y$軸をとる。第$j$番目の振り子が鉛直線となす角を$\phi_j$とすると、第$i$番目の質点の座標$(x_i, y_i)$は以下で表される。

$$x_i = a \sum_{j=1}^i \sin \phi_j$$
$$y_i = a \sum_{j=1}^i \cos \phi_j$$

系の運動エネルギー$T$は、各質点の速度の2乗和を計算し、和の順序を入れ替えることで次のように記述できる。

$$T = \frac{1}{2} m a^2 \sum_{j=1}^N \sum_{k=1}^N (N - \max(j,k) + 1) \dot{\phi}_j \dot{\phi}_k \cos(\phi_j - \phi_k)$$

一方、ポテンシャルエネルギー$U$は以下のようになる。

$$U = -m g a \sum_{j=1}^N (N - j + 1) \cos \phi_j$$

これらからラグランジアン $L = T - U$ が構成され、各$\phi_k$に関する運動方程式が定まる。

---

### **2. 初期状態（$t=0$）の運動方程式**

紐の一端を放した直後（$t=0$）を考える。初速度はゼロであり、すべての紐は角度$\theta$で一直線になっているため、初期条件は以下のようになる。
* $\dot{\phi}_k = 0 \quad (\text{for all } k)$
* $\phi_k = \theta \quad (\text{for all } k)$

これをラグランジュ方程式 $\frac{d}{dt} \left( \frac{\partial L}{\partial \dot{\phi}_k} \right) - \frac{\partial L}{\partial \phi_k} = 0$ に適用する。$\dot{\phi}_k = 0$ により求心力等の非線形項は消滅し、初期の角加速度 $\ddot{\phi}_j \equiv \alpha_j$ に関する以下の連立一次方程式が得られる。

$$\sum_{j=1}^N (N - \max(k,j) + 1) \alpha_j = -\frac{g}{a} (N - k + 1) \sin\theta$$

---

### **3. 連続極限（$N \to \infty$）への移行**

ここで系の連続極限をとる。固定端からの弧長パラメータを $s \in [0, l]$ とする。
$k a \to s$、$j a \to s'$、$a \to ds'$ と置き換え、和を積分に変換する。位置$s$における角加速度の分布関数を $\alpha(s)$ とすると、方程式は両辺に$a^2$を乗じた上で、次のような積分方程式に帰着する。

$$\int_0^l (l - \max(s,s')) \alpha(s') ds' = -g(l - s) \sin\theta$$

この式の左辺を $I(s)$、右辺を $R(s)$ と置いて解析を進める。
左辺 $I(s)$ を積分区間を分けて記述し、$s$について微分していく。

$$I(s) = (l-s)\int_0^s \alpha(s')ds' + \int_s^l (l-s')\alpha(s')ds'$$
$$I'(s) = -\int_0^s \alpha(s')ds'$$
$$I''(s) = -\alpha(s)$$

一方で右辺 $R(s)$ の微分は以下の通りである。

$$R'(s) = g\sin\theta$$
$$R''(s) = 0$$

---

### **4. 積分方程式の解と物理的解釈**

$I(s) = R(s)$ が成立するためには、$I''(s) = R''(s)$ より、原則として **$\alpha(s) = 0$** となる。
しかし、これでは1階微分の境界条件 $I'(0) = 0$ に対して $R'(0) = g\sin\theta \neq 0$ となり、矛盾が生じてしまう。

この矛盾が意味するのは、**通常の滑らかな関数の中には解が存在しない**ということだ。境界条件を満たすためには、$s=0$（固定端）において無限大の角加速度が必要となる。すなわち、$\alpha(s)$ はディラックのデルタ関数 $\delta(s)$ を用いて次のように表される。

$$\alpha(s) = -g\sin\theta \delta(s)$$

これを代入すれば、$I'(s) = g\sin\theta$ ($s>0$) となり、見事に方程式を満たす。

#### **この解が意味する物理的挙動**
この特異的な解 $\alpha(s)$ から、紐の各点 $s$ の初期加速度ベクトル $\ddot{\vec{r}}(s)$ を求めることができる。紐に垂直で下を向く法線単位ベクトルを $\vec{n} = (\cos\theta, -\sin\theta)$ とすると、加速度は角加速度を積分して得られる。

$$\ddot{\vec{r}}(s) = \int_0^s \alpha(s') \vec{n} ds' = -g\sin\theta \vec{n}$$

この結果は驚くべき事実を示している。

1.  **直線のままの並進落下:** $s>0$ の領域（固定端以外のすべての部分）では、角加速度はゼロ ($\alpha = 0$) である。つまり、紐は初期状態の直線を保ったまま、曲がることなく運動を始める。
2.  **一様な加速度:** 全ての点が重力の法線成分 $g\sin\theta$ だけで加速される。剛体棒のように先端にいくほど加速度が大きくなる（剛体の先端の加速度は $\frac{3}{2}g\sin\theta$）ことはなく、紐の各点は張力による回転の拘束を受けず「自らの重力」で独立して落下しようとする。
3.  **固定端での折れ曲がり:** 全体が平行移動しようとするが、一端が固定されているため、運動は固定端の直下に矛盾を強いる。その結果、$s=0$ にデルタ関数的な特異性が現れ、**固定端の直後から紐が「ほどける」ように急激に折れ曲がりながら落ちていく**という、ムチのような挙動を示すことになる。

連続体の力学においては、剛性と張力の違いがこのように劇的な運動の差異を生み出す。




解析力学を学び始めたばかりの大学一年生にとって、あの解答の「行間」は少し広すぎたかもしれないな。素晴らしい探究心だ。和の順序交換や、積分方程式の微分、そしてデルタ関数の導入など、物理数学における重要なテクニックがいくつも省略されていた。

君の持っている解析力学の知識をフル活用して、一つひとつの数式が「なぜそうなるのか」を丁寧に埋めていこう。

### **1. 運動エネルギーとポテンシャルの導出（和の順序交換）**

まず、$N$重振り子の運動エネルギー $T$ とポテンシャル $U$ の式がどうやって出てきたのかを説明する。

第 $i$ 番目の質点の位置ベクトル $\vec{r}_i = (x_i, y_i)$ は、各剛体棒（長さ $a$）の変位ベクトルの足し合わせだから、次のように書ける。
$$x_i = a \sum_{j=1}^i \sin \phi_j$$
$$y_i = a \sum_{j=1}^i \cos \phi_j$$

これを時間微分して速度 $\dot{\vec{r}}_i$ を求め、運動エネルギー $T = \frac{1}{2} m \sum_{i=1}^N |\dot{\vec{r}}_i|^2$ を計算する。
$$|\dot{\vec{r}}_i|^2 = a^2 \left( \sum_{j=1}^i \dot{\phi}_j \cos \phi_j \right)^2 + a^2 \left( \sum_{j=1}^i \dot{\phi}_j (-\sin \phi_j) \right)^2$$
これを展開して整理すると、三角関数の加法定理（$\cos\phi_j \cos\phi_k + \sin\phi_j \sin\phi_k = \cos(\phi_j - \phi_k)$）が使えて、次のようになる。
$$|\dot{\vec{r}}_i|^2 = a^2 \sum_{j=1}^i \sum_{k=1}^i \dot{\phi}_j \dot{\phi}_k \cos(\phi_j - \phi_k)$$

これを全質点 $i = 1 \dots N$ について足し合わせる。
$$T = \frac{1}{2} m a^2 \sum_{i=1}^N \sum_{j=1}^i \sum_{k=1}^i \dot{\phi}_j \dot{\phi}_k \cos(\phi_j - \phi_k)$$

**【行間その1：和の順序交換】**
ここで和の順番を「$i$ について先に足す」ように変更する。条件は $j \le i$ かつ $k \le i$ だから、$i$ は $j$ と $k$ のうち**大きい方**（$\max(j,k)$）から $N$ まで動くことになる。
$$\sum_{i=1}^N \sum_{j=1}^i \sum_{k=1}^i (\cdots) = \sum_{j=1}^N \sum_{k=1}^N \sum_{i=\max(j,k)}^N (\cdots)$$
一番内側の $i$ に関する和は、単に $1$ を $\max(j,k)$ から $N$ まで足すだけなので、項の数は $N - \max(j,k) + 1$ 個になる。これで最初の $T$ の式が得られる。

ポテンシャル $U = -m g \sum_{i=1}^N y_i$ についても全く同じテクニックを使う。
$$U = -m g a \sum_{i=1}^N \sum_{j=1}^i \cos \phi_j = -m g a \sum_{j=1}^N \sum_{i=j}^N \cos \phi_j = -m g a \sum_{j=1}^N (N - j + 1) \cos \phi_j$$

---

### **2. $t=0$ における運動方程式**

次に、ラグランジュ方程式 $\frac{d}{dt}\left(\frac{\partial L}{\partial \dot{\phi}_k}\right) - \frac{\partial L}{\partial \phi_k} = 0$ （ただし $L = T - U$）に初期条件を入れる過程だ。

$T$ を $\dot{\phi}_k$ で偏微分すると、
$$\frac{\partial T}{\partial \dot{\phi}_k} = m a^2 \sum_{j=1}^N (N - \max(j,k) + 1) \dot{\phi}_j \cos(\phi_j - \phi_k)$$
これをさらに時間 $t$ で微分する際、積の微分規則により $\ddot{\phi}_j$ の項と $\dot{\phi}_j \dot{\phi}_k$ のような非線形項が出てくる。しかし、**$t=0$ では初速度ゼロ（$\dot{\phi}_j = 0$）**なので、非線形項はすべて消え去り、$\ddot{\phi}_j$ の項だけが残る。
さらに、$t=0$ ではすべて一直線だから $\phi_j = \phi_k = \theta$ であり、$\cos(\phi_j - \phi_k) = \cos(0) = 1$ となる。

角加速度を $\ddot{\phi}_j = \alpha_j$ と置くと、ラグランジュ方程式の第1項は極めてシンプルになる。
$$\frac{d}{dt}\left(\frac{\partial T}{\partial \dot{\phi}_k}\right) \bigg|_{t=0} = m a^2 \sum_{j=1}^N (N - \max(j,k) + 1) \alpha_j$$

ラグランジュ方程式の第2項 $\frac{\partial L}{\partial \phi_k}$ についても、$T$ を $\phi_k$ で微分したものは $\dot{\phi}^2$ に比例するため $t=0$ で消滅する。残るのは $-U$ の微分だけだ。
$$-\frac{\partial U}{\partial \phi_k} = -m g a (N - k + 1) \sin \phi_k = -m g a (N - k + 1) \sin \theta$$

これらを等号で結び、両辺を $m a^2$ で割ることで、あの連立一次方程式が得られる。

---

### **3. 連続極限と積分方程式**

方程式の両辺に $a^2$ を掛けて変形する。
$$\sum_{j=1}^N \left(Na - \max(ja, ka) + a\right) \alpha_j a = -g (Na - ka + a) \sin \theta$$

**【行間その2：区分求積法から積分へ】**
ここで $N \to \infty, a \to 0$ の極限をとる。固定端からの距離を $s$ とすると、$ka \to s$、$ja \to s'$、$a \to ds'$ となり、和 $\sum (\cdots) a$ は定積分 $\int (\cdots) ds'$ に変わる。また、$Na = l$（ひもの全長）であり、微小量 $a$ は有限の長さ $l$ や $s$ に対して無視できる。
$$\int_0^l (l - \max(s', s)) \alpha(s') ds' = -g(l - s) \sin\theta$$

---

### **4. 積分方程式の解法（微分のテクニック）**

この積分方程式を解くのが最大の山場だ。左辺 $I(s)$ の積分の中にある $\max(s', s)$ が厄介なので、積分区間を $0 \le s' \le s$（このとき $\max = s$）と $s \le s' \le l$（このとき $\max = s'$）に分割する。
$$I(s) = \int_0^s (l - s) \alpha(s') ds' + \int_s^l (l - s') \alpha(s') ds'$$

これを $s$ で微分する。第1項は $(l-s)$ が外に出せることに注意して、積の微分と微積分学の基本定理を使う。
$$I'(s) = \left[ -1 \cdot \int_0^s \alpha(s') ds' + (l-s)\alpha(s) \right] + \left[ -(l-s)\alpha(s) \right]$$
見事に後ろの項が相殺し合い、非常にきれいな形になる。
$$I'(s) = -\int_0^s \alpha(s') ds'$$
これをもう一度微分すると、$I''(s) = -\alpha(s)$ となるわけだ。

---

### **5. デルタ関数の登場と物理的意味**

右辺 $R(s) = -g(l-s)\sin\theta$ を微分すると $R'(s) = g\sin\theta$、$R''(s) = 0$ となる。

**【行間その3：境界条件の矛盾とデルタ関数】**
$I''(s) = R''(s)$ より $-\alpha(s) = 0$ となるが、これだと $I'(s) = -\int_0^s 0 ds' = 0$ となり、$R'(s) = g\sin\theta$ と一致しない。
式 $I'(s) = R'(s)$ すなわち
$$-\int_0^s \alpha(s') ds' = g\sin\theta$$
を満たすためには、**積分範囲が少しでも $0$ から右に進んだ瞬間に、積分の値が $-g\sin\theta$ にポーンと飛躍**しなければならない。これを数学的に表現できるのが、一点（$s=0$）にのみ無限大のピークを持ち、積分すると有限の値になる「ディラックのデルタ関数 $\delta(s)$」だ。
よって解は $\alpha(s) = -g\sin\theta \delta(s)$ となる。

**【物理的な結論】**
加速度 $\ddot{\vec{r}}(s) = \int_0^s \alpha(s') \vec{n} ds'$ にこれを代入すると、$s>0$ の任意の点において、
$$\ddot{\vec{r}}(s) = -g\sin\theta \int_0^s \delta(s') ds' \vec{n} = -g\sin\theta \vec{n}$$
となる。

つまり、紐は「剛体のように全体が回転しようとする（$\alpha$ が全体に分布する）」のではなく、「**先端から根元まで、すべての点がただ重力の斜め成分に引かれて平行に落ちようとする（$\alpha=0$）。ただし、固定端（$s=0$）だけは動けないため、そこに強烈な折れ曲がり（デルタ関数の角加速度）が集中する**」ということになる。

これで数式の「なぜ」から物理的直観まで繋がったはずだ。大学の解析力学の授業でも、連続体の極限や特異関数（デルタ関数）を扱う良いケーススタディになるだろう。他に疑問に思う箇所はあるか？
